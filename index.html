Privacy Policy / Gizlilik Politikasi

Last updated / Son guncelleme: March 31, 2025

Electricity Tycoon ("App") values the privacy of its users. This privacy policy explains what data is collected when you use our app, how it is used, and how it is protected.

Electricity Tycoon ("Uygulama") olarak kullanicilarin gizliligine buyuk onem veriyoruz. Bu gizlilik politikasi, uygulamamizi kullanirken hangi verilerin toplandigini, nasil kullanildigini ve nasil korundugunuaciklamaktadir.

1. Collected Data / Toplanan Veriler

Our app collects the following data through Firebase services:

Data Type Purpose Stored Where
Anonymous user ID Cloud save & progress sync Firebase Authentication
Google account info (if linked) Optional sign-in for cross-device sync Firebase Authentication
Game progress data Cloud save backup Firebase Firestore
Gameplay analytics Game balance improvement Firebase Analytics

2. Analytics Events / Analitik Olaylari

We collect anonymous gameplay events to improve game balance:

  • Chapter progression (which level the player reached)
  • Equipment purchases
  • Session start/end
  • Premium purchases

These events do not contain personal information. They are aggregated and used solely for improving the game experience.

3. Third-Party Services / Ucuncu Taraf Hizmetleri

Our app uses the following third-party services:

  • Firebase Authentication - for anonymous and Google sign-in (Firebase Privacy Policy)
  • Firebase Firestore - for cloud save storage
  • Firebase Analytics - for anonymous gameplay analytics

We do not use any advertising SDKs. We do not sell or share user data with third parties for advertising purposes.

4. Children's Privacy / Cocuklarin Gizliligi

Our app is suitable for all age groups. We do not knowingly collect personal data from children under 13. The anonymous authentication system does not require any personal information to use the app.

5. Data Security / Veri Guvenligi

All data transmitted between the app and Firebase servers is encrypted via HTTPS/TLS. Game progress data stored locally on the device is not accessible to other apps.

6. Data Retention / Veri Saklama

Your data is retained as long as your account exists. You can delete your account and all associated data at any time:

  • In-app: Settings > Delete My Account
  • Web: Account Deletion Page

7. Your Rights / Haklariniz

You have the right to:

  • Request deletion of your account and all associated data
  • Use the app without linking a Google account (anonymous mode)
  • Opt out of cloud saves by not signing in with Google

8. Changes / Degisiklikler

We may update this privacy policy from time to time. Changes will be published on this page. Continued use of the app constitutes acceptance of the updated policy.
